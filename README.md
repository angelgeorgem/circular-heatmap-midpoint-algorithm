# Circular Heatmap using Midpoint Circle Algorithm

## 📌 Overview
This project demonstrates a **circular heatmap simulation** using the **Midpoint Circle Drawing Algorithm** combined with **scaling transformations** and **animation in Python**.

The heatmap visualizes how heat propagates outward from a central source using expanding circular rings.

---

## 🚀 Features
- Midpoint Circle Algorithm implementation
- Scaling transformation for dynamic expansion
- Animated heat propagation
- Color-based heat intensity visualization
- Gradient heatmap using distance normalization
- Real-time status labels (High, Medium, Low heat)

---

## 🧠 Concepts Used
- Computer Graphics Algorithms
- Midpoint Circle Drawing Algorithm
- Geometric Scaling Transformation
- Matplotlib Animation

---

## 🛠️ Technologies Used
- Python
- Matplotlib


---

## 📊 How It Works
1. The **Midpoint Circle Algorithm** generates boundary points of a circle.
2. These points are **scaled progressively** to simulate expansion.
3. Heat intensity is calculated based on **distance from the center**.
4. Colors are mapped using a **heat colormap (`hot`)**.
5. Animation shows **heat spreading over time**.
